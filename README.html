Key findings
Most water-intensive per acre
7.02 AF/ac/yr
Google The Dalles — more than any irrigated crop; 5.3× site precipitation; 1/3 of city supply for pop. ~16,200
Best-in-class efficiency
Near 0 AF/yr
Microsoft Quincy (QWRU closed-loop), Switch Citadel (air), Google Hamina (seawater) — geography + technology = zero water burden
Counterintuitive finding
0.60 AF/ac/yr
Dense urban residential (apartments, Q5) — less water per acre than semi-rural residential (1.80) or most data centers
Largest disclosure gap
~75%
Facilities in dataset with no metered water data — estimated from MW capacity or undisclosed. Only Google discloses site-level figures proactively.
How to use locally
Open in browser: Double-click index.html. All interactivity works offline. D3.js loads from cdnjs CDN and Google Fonts require WiFi on first load — otherwise fully self-contained.

Publish to GitHub Pages: This folder is the complete site. Push all four files to a repo's root, enable Pages on the main branch, and your URL will be https://[username].github.io/[repo-name]/.

Add facilities: Open index.html in a text editor, find const FACILITIES = [, add entries following the same object structure. Update data_manifest.csv in parallel.

Peer review: See methodology.html §9 for step-by-step reproducibility instructions, including how to obtain primary sources and validate each confidence tier.
